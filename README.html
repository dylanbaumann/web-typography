Web Typography

Example of Beautiful Web Typography including fractions, ligatures, small capitals, and true italics.

Important Info

This project includes fonts which were paid for and owned by me. Do not utilize these fonts for purposes outside of this specific repo unless you have purchased licensing for your own use. The font is named Calendas Plus (you can get it here) and it is one of the nicest fonts I’ve ever come across and it’s dirt-cheap, so please pick up your own copy before using on other projects.
